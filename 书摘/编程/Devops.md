# Devops 十年

## DevOps 和工具链简介

#### DevOps 

通过自动化和架构变更，从而提升研发效率，节约成本

DevOps 包含一下因素：

- 流程自动化
- 架构微服务
- 度量
- 文化和意识
- 分享

#### DevOps 工具链

流程自动化是实现DevOps的基础，通常通过工具或者工具链进行实现的



一个完整的 端到端软件开发生命周期 SOLC 一般包含：

- 项目管理
- 源代码管理
- 持续集成
- 代码质量分析
- 单元测试
- 编译构造
- 制品管理
- 自动化测试
- 自动化发布
- 监控预警
- 日志分析
